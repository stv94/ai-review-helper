# Changelog

All notable changes to **AI Merge Request Reviewer** will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] — 2026-03-13

### Added
- Guided, step-by-step AI review walkthrough for GitLab Merge Requests
- Inline and split diff viewer with syntax highlighting
- Inline commenting on specific diff lines
- General MR-level comments (add and delete)
- MR approval and approval revocation
- AI-generated review narrative: overview, logical change blocks, critical analysis
- Multi-language UI: English, Russian, German, French, Spanish, Portuguese, Chinese, Japanese
- LLM provider support: OpenAI, OpenRouter, and custom OpenAI-compatible endpoints
- Configurable diff chunk size to fit any model's context window
- Settings panel with connection testing
- MR metadata display (title, author, branch, status)
